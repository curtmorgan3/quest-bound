# Fight or Fright - Test Content

This directory contains test data for the "Fight or Fright" Halloween-themed dungeon crawler ruleset. This content can be used for development and testing of the import/export functionality.

## Content Overview

### Ruleset: Fight or Fright
A Halloween-themed dungeon crawler where brave adventurers navigate haunted houses, battle supernatural creatures, and uncover dark secrets.

### Files Included

1. **metadata.json** - Ruleset metadata and export information
2. **charts.json** - Four comprehensive charts (600 total entries):
   - **Monsters Chart** (300 entries) - Supernatural creatures with stats, abilities, and weaknesses
   - **Animals Chart** (100 entries) - Various creatures that may aid or hinder players
   - **Item Shop Chart** (150 entries) - Items with rarity levels and pricing
   - **Floor Plans Chart** (50 entries) - Mansion layouts with room details and danger levels

## Final Counts

- **Attributes**: 8 character stats
- **Actions**: 18 spooky actions
- **Items**: 238 Halloween-themed items
- **Charts**: 600 total entries across 4 charts
  - Monsters: 300 supernatural creatures
  - Animals: 100 various creatures
  - Shop Items: 150 items with rarity and pricing
  - Floor Plans: 50 mansion layouts

## Chart Details

### Monsters Chart (300 entries)
Each monster includes:
- Health (20-200), Fear Level (10-100), Damage (5-50)
- Three random abilities (Teleportation, Invisibility, Possession, etc.)
- Weaknesses (Silver, Holy Water, Salt, Garlic, etc.)
- Haunted locations (Attic, Basement, Graveyard, etc.)

### Animals Chart (100 entries)
Each animal includes:
- Size (1-50), Speed (1-100), Intelligence (1-50)
- Three behaviors (Hunting, Flying, Territorial, etc.)
- Habitat information
- Notes on potential assistance or hindrance

### Item Shop Chart (150 entries)
Each shop item includes:
- Rarity levels: Common, Uncommon, Rare, Epic, Legendary
- Price based on rarity multipliers (1x to 25x base price)
- Stock quantities (rarer items have less stock)
- Category classification (Weapons, Armor, Consumables, Tools, Magic Items)

### Floor Plans Chart (50 entries)
Each floor plan includes:
- Mansion name and floor type
- Total area (500-2500 sq ft)
- Number of rooms (3-10)
- Exit count and danger level
- Trap and treasure indicators
- Detailed room listings with sizes and secret passages

## Usage

These files can be imported into Quest Bound using the import functionality to create a complete "Fight or Fright" ruleset for testing and development. The content provides a rich, immersive Halloween-themed gaming experience with comprehensive character progression, combat mechanics, supernatural encounters, and detailed exploration elements.
